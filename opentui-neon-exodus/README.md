# opentui-neon-exodus

Terminal showcase for the Neon Exodus interface system.

Run locally:

```bash
bun install
bun run start
```

Typecheck:

```bash
bun run typecheck
```

Controls:

- `1-5` or `h` / `l` switch sections
- Arrow keys move selection through the active demo grid
- Mouse hover selects widgets, and double-click or right-click maximizes them
- `Enter` or `f` opens the selected widget fullscreen with audio
- `Esc` or `t` returns to the tiled view
- `+` / `-` adjust volume
- `q` exits

The full project overview, screenshots, and web build notes are in the repo root [`README.md`](../README.md).
