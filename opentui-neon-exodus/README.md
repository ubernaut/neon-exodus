# opentui-neon-exodus

Terminal showcase for the Neon Exodus interface system.

Run locally:

```bash
bun install
bun run start
```

Typecheck:

```bash
bun run typecheck
```

Controls:

- `1-4` or `h` / `l` switch sections
- Arrow keys navigate the 3D widget deck
- `Enter` or `f` opens the selected widget fullscreen
- `Esc` or `t` returns to the tiled deck
- `q` exits

The full project overview, screenshots, and web build notes are in the repo root [`README.md`](../README.md).
